npm init -y
این دستور یک فایل package.json جدید در پوشه server ایجاد می‌کند. این فایل اطلاعات پروژه مانند وابستگی‌ها (dependencies) و تنظیمات دیگر را نگهداری می‌کند.



نصب وابستگی‌ها: بعد از ایجاد پروژه، شما نیاز دارید تا برخی از وابستگی‌ها را نصب کنید. برای مثال:

Express: فریم‌ورک محبوب NodeJS برای ساخت API‌ها و سرورها.

pg: کتابخانه برای ارتباط با پایگاه داده PostgreSQL.

برای نصب این وابستگی‌ها، از دستور زیر استفاده کنید:

npm install express pg


ساخت و تنظیم پایگاه داده PostgreSQL: در این قسمت، شما باید یک پایگاه داده PostgreSQL بسازید و برخی اطلاعات اولیه را در آن قرار دهید. مثلاً می‌توانید یک جدول برای ذخیره Todoها بسازید.

ساخت سرور NodeJS با Express: در مرحله بعد، باید یک سرور ساده با استفاده از Express بسازید که بتواند درخواست‌ها را دریافت کند و به پایگاه داده PostgreSQL متصل شود.

آزمایش بک‌اند: پس از راه‌اندازی سرور و پایگاه داده، می‌توانید از یک ابزار REST Client (مثل Postman یا Insomnia) برای ارسال درخواست‌ها به سرور و بررسی عملکرد بک‌اند استفاده کنید.
